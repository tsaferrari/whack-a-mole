# whack-a-mole
Simple whack a mole game using java swing GUI

Whack-a-mole game using Swing. Game will allow a player to play the game for 20 seconds before the game stops. 

When player clicks on a “mole” with its head “up”, the player gets a point. 

This game can be played by many users at the same time and many games can run concurrently on the same machine.
